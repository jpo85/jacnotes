## gestione della ILO HPE tramite ssh e commandline

**comandi per cambiare i dns server**

```bash
sh -c aes256-cbc -oKexAlgorithms=diffie-hellman-group1-sha1 -o HostKeyAlgorithms=ssh-dss hpadmin@192.168.1.10
show map1/dnsserver1
show map1/dnsserver2
set map1/settings1/DNSSettings1 DHCPOptionToUse=no
set map1/settings1/DNSSettings1 DNSServerAddress=10.10.10.1
set map1/dnsserver1 AccessInfo=10.10.10.1
set map1/dnsserver2 AccessInfo=10.10.10.2
```
