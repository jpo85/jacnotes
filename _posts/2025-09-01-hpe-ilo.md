Se vi trovate a dover cambiare i dns dell'interfaccia della ILO dei server HPE tramite commandline, questa è la senquenza dei comandi da eseguire.
Il primo comando serve ad effettuare la sessione in ssh

```bash
ssh -c aes256-cbc -oKexAlgorithms=diffie-hellman-group1-sha1 -o HostKeyAlgorithms=ssh-dss hpadmin@192.168.1.10
```
Una volta effettuato il login vi troverete nella commandline della ILO, con questo set di istruzioni da eseguire
```
show map1/dnsserver1
show map1/dnsserver2
set map1/settings1/DNSSettings1 DHCPOptionToUse=no
set map1/settings1/DNSSettings1 DNSServerAddress=10.10.10.1
set map1/dnsserver1 AccessInfo=10.10.10.1
set map1/dnsserver2 AccessInfo=10.10.10.2
```
